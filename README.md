I den här labben har jag använt mig av C kod och python för att skapa ett program som validerar IP adresser och brandväggsportar
Jag har använt mig utav bibloteken stdio.h stdlib.h string.h ctype.h för att kunna koda i C.

det är strukturerat såhär
- IP‑validering
Kontrollerar att en given IP‑adress följer korrekt innehåll.
- Portvalidering
Säkerställer att angiven port ligger inom giltigt intervall för brandväggsportar.
- Loggning
Föregående val sparas och kan visas för användaren.
- Avslut
Programmet stängs ned på ett kontrollerat sätt.
